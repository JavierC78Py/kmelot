Módulo de WhatsApp

Sistema Kmelot - Santa Med

Integración de mensajería WhatsApp con Oracle APEX para una comunicación eficiente

Introducción

Kmelot permite la integración de servicios de mensajería directamente desde la plataforma, facilitando la comunicación con clientes y personal mediante la API no oficial de WhatsApp.

Características Principales

Envío de mensajes

Comunicación individual y masiva a través de WhatsApp

Plantillas predefinidas

Mensajes estructurados con formatos preaprobados

Bot para gestión de reservas.

Confirmación de reservas que se actualizan al erp.

Historial de mensajes

Registro completo de todas las comunicaciones

Acceso al Sistema

Para acceder al módulo de WhatsApp, los usuarios deben autenticarse en el sistema Kmelot mediante sus credenciales asignadas:

Pantalla de Login - Kmelot Santa Med

Pantalla de Chats

La pantalla de chats permite gestionar todas las conversaciones de WhatsApp. Para acceder a esta funcionalidad:

  1. Navegar al menú Mensajes WhatsApp
  2. Seleccionar Procesos
  3. Hacer clic en Chats

Funcionalidades de la Pantalla de Chat

  • Lista de chats: Muestra todas las conversaciones activas en el lado izquierdo
  • Historial de chat: Visualización completa de la conversación en el área central derecha
  • Información del contacto: En la parte superior se muestran los datos del número/chat seleccionado
  • Área de escritura: En la parte inferior para redactar mensajes
  • Botones de acción:
    • Icono de enviar (➤): Envía el mensaje escrito
    • Icono de plantilla (📋): Muestra lista desplegable de plantillas para envío rápido
    • Icono de adjuntar (📎): Permite enviar archivos

Opciones parte superior derecha.

Lista de departamentos, sesiones activas, y boton de nuevo mensaje.
